# omni

`omni` — header-only C++23 библиотека для Windows-oriented low-level разработки: inspection модулей и экспортов, lazy import, syscall wrappers, shared user data и compile-time hashing.

## Основные возможности

- доступ к loader list как к range через `omni::modules`;
- работа с PE export table через `omni::module_exports` и `omni::module_export`;
- поиск экспортов по имени, хешу и ordinal;
- вызов WinAPI через `omni::lazy_import` и `omni::lazy_importer`;
- вызов NT syscalls через `omni::syscall` и `omni::syscaller`;
- чтение `KUSER_SHARED_DATA` через `omni::shared_user_data()`;
- compile-time hash типы `omni::fnv1a32`, `omni::fnv1a64` и `omni::default_hash`.
