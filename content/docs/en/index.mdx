# omni

`omni` is a header-only C++23 library for Windows-oriented low-level development: module and export inspection, lazy import, syscall wrappers, shared user data, and compile-time hashing.

## Key capabilities

- access the loader list as a range via `omni::modules`;
- work with the PE export table via `omni::module_exports` and `omni::module_export`;
- look up exports by name, hash, and ordinal;
- call WinAPI via `omni::lazy_import` and `omni::lazy_importer`;
- call NT syscalls via `omni::syscall` and `omni::syscaller`;
- read `KUSER_SHARED_DATA` via `omni::shared_user_data()`;
- use compile-time hash types `omni::fnv1a32`, `omni::fnv1a64`, and `omni::default_hash`.

## Who this documentation is for

This site documents only the public API available to library users. Internal details from `omni/detail/*` are not documented as a supported contract.
